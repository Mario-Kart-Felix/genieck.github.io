    了解 Epic Games 如何利用 AWS 向全球超過 2 億玩家提供 Fortnite。
    進一步了解 

    在雲端上建立基礎

    Epic Games 自 2012 年以來一直使用 AWS，現在全部投入 AWS 雲端，該公司在 AWS 上執行其全球遊戲伺服器叢集、後端平台系統、資料庫、網站、分析管道和處理系統。

    一夜之間就引起轟動

    2017 年，Epic Games 推出了 Fortnite，這是一款跨平台的多人遊戲，一夜之間就引起轟動。在第一年，Fortnite 的玩家群增長了 100 多倍，全球達到 2 億位玩家。

    打破極限
