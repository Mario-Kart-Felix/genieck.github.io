Amazon S3

Penyimpanan objek yang dibangun untuk menyimpan dan mengambil sejumlah data dari mana pun

Memulai dengan Amazon S3
Minta informasi selengkapnya

Amazon Simple Storage Service (Amazon S3) adalah layanan penyimpanan objek yang menawarkan skalabilitas, ketersediaan data, keamanan, dan kinerja terdepan dalam industri. Ini berarti pelanggan dari segala ukuran dan industri dapat menggunakannya untuk menyimpan dan melindungi data sebanyak apa pun untuk berbagai kasus pengunaan, seperti situs web, aplikasi perusahaan, perangkat IoT, dan analisis big data. Amazon S3 memberikan fitur manajemen yang mudah sehingga Anda dapat mengatur data dan mengonfigurasi kontrol akses yang disetel dengan baik guna memenuhi kebutuhan bisnis, organisasional, dan kepatuhan tertentu. Amazon S3 didesain untuk ketahanan data 99,999999999% (11 9) karena secara otomatis membuat dan menyimpan salinan dari semua objek S3 di beberapa sistem.

Keuntungan

Kinerja, skalabilitas, ketersediaan, dan ketahanan terdepan dalam industri

Sesuaikan sumber daya penyimpanan Anda naik dan turun untuk memenuhi permintaan yang berfluktuasi, tanpa pelunasan investasi di awal atau siklus penyediaan sumber daya. Amazon S3 didesain untuk ketahanan data 99,999999999% (11 9) karena secara otomatis membuat dan menyimpan salinan dari semua objek S3 di beberapa sistem. Ini berarti data Anda tersedia ketika dibutuhkan dan dilindungi dari kegagalan, kesalahan, dan ancaman.
Pelajari tentang ketahanan data S3 »

Jangkauan luas kelas penyimpanan yang hemat biaya

Menghemat biaya tanpa mengorbankan kinerja dengan menyimpan data di semua Kelas Penyimpanan S3, yang mendukung level akses data berbeda pada rate tertentu. Anda dapat menggunakan Analisis Kelas Penyimpanan S3 untuk menemukan data yang harus dipindahkan ke kelas penyimpanan berbiaya rendah berdasarkan pola akses, dan mengonfigurasikan kebijakan Siklus Aktif S3 untuk menjalankan transfer. Anda juga dapat menyimpan data dengan pola akses yang berubah atau tidak diketahui di Pengkategorian Cerdas S3, yang memeringkatkan objek berdasarkan pada pola akses yang berubah dan secara otomatis menghasilkan penghematan biaya.
Baca selengkapnya tentang Kelas Penyimpanan S3 »

Kemampuan keamanan, kepatuhan, dan audit tak tertandingi

Simpan data Anda di Amazon S3 dan lindungi dari akses tidak berwenang dengan fitur enkripsi dan akses alat manajemen. S3 adalah satu-satunya layanan penyimpanan objek yang memungkinkan Anda memblokir akses publik ke semua objek Anda pada bucket atau tingkat akun dengan Akses Publik Blok S3. S3 mengelola program kepatuhan, seperti PCI-DSS, HIPAA/HITECH, FedRAMP, EU Data Protection Directive, dan FISMA, untuk membantu Anda memenuhi persyaratan hukum. AWS juga mendukung beberapa kemampuan audit untuk mengawasi permintaan akses ke sumber daya S3 Anda.
Baca selengkapnya tentang keamanan dan kepatuhan S3 »

Mudah mengelola data mengakses kontrol

S3 memberi Anda kemampuan kuat untuk mengelola akses, biaya, replikasi, dan perlindungan data. S3 Access Points memudahkan pengelolaan akses data dengan izin spesifik untuk aplikasi Anda dengan menggunakan set data bersama. Fitur-fitur Replikasi S3 mengelola replikasi data di dalam wilayah atau ke wilayah lain. S3 Batch Operations membantu mengelola perubahan skala besar di miliaran objek. Karena S3 cocok dengan AWS Lambda, Anda dapat mencatat aktivitas, menentukan peringatan, dan mengotomatiskan alur kerja, tanpa perlu mengelola infrastruktur tambahan.
Pelajari selengkapnya tentang fitur manajemen penyimpanan S3 »

Layanan kueri yang siap digunakan untuk analisis

Jalankan analisis big data di semua objek S3 (dan set data lainnya di AWS) dengan layanan kueri yang siap digunakan kami. Gunakan Amazon Athena untuk kueri data S3 dengan ekspresi SQL standar dan Amazon Redshift Spectrum untuk menganalisis data yang disimpan ke gudang data AWS dan sumber daya S3. Anda juga dapat menggunakan S3 Select untuk mengambil subset data objek--bukan keseluruhan objek, dan meningkatkan kinerja kueri hingga 400%
Pelajari selengkapnya tentang kueri yang siap digunakan »

Layanan penyimpanan cloud paling didukung

Simpan dan lindungi data di Amazon S3 dengan bekerja dengan mitra dari AWS Partner Network (APN) — komunitas teknologi terbesar dan penyedia layanan cloud konsultasi. APN mengenali mitra migrasi yang mentransfer data ke Amazon S3 dan mitra penyimpanan yang menawarkan solusi terintegrasi S3 untuk penyimpanan utama, pencadangan dan pemulihan, arsip, dan pemulihan bencana. Anda juga dapat membeli solusi terintegrasi AWS langsung dari AWS Marketplace, yang mencantumkan hingga lebih dari 250 penawaran khusus penyimpanan.
Pelajari tentang APN dan AWS Marketplace »

Cara kerja — S3 Access Points

Amazon S3 Access Points memudahkan Anda mengelola akses sesuai skala untuk aplikasi dengan menggunakan set data yang dibagikan pada S3. Dengan S3 Access Points, Anda kini dapat dengan mudah membuat ratusan titik akses per bucket, mewakili cara baru penyediaan akses ke set data bersama. Access Points menyediakan jalur khusus ke bucket, dengan nama host unik dan kebijakan akses yang memberlakukan izin khusus dan kontrol jaringan untuk setiap permintaan yang dibuat melalui titik akses.

Kasus penggunaan

Pencadangan dan pemulihan

Membangun solusi yang dapat diskalakan, tahan lama, dan melindungi pencadangan dan pemulihan dengan Amazon S3 dan layanan AWS lainnya, sepertiS3 Glacier, Amazon EFS, dan Amazon EBS, untuk memaksimalkan atau mengganti kemampuan di lokasi yang sudah ada. Mitra AWS dan APN dapat membantu Anda memenuhi Recovery Time Objectives (RTO), Recovery Point Objectives (RPO), dan persyaratan kepatuhan. Dengan AWS, Anda dapat mencadangkan data yang sudah ada di AWS Cloud atau menggunakan AWS Storage Gateway, layanan penyimpanan hybrid, untuk mengirimkan cadangan data di lokasi ke AWS.
Pelajari selengkapnya tentang pencadangan & pemulihan »

Pemulihan Bencana (DR)

Melindungi data, aplikasi, dan sistem TI penting yang berjalan di AWS Cloud atau lingkungan di lokasi tanpa mengenakan biaya pada lokasi fisik kedua. Dengan penyimpanan Amazon S3, Replikasi Lintas Wilayah S3, dan layanan komputasi, jaringan, dan database AWS lain, Anda dapat membuat arsitektur DR dengan cepat dan mudah pulih dari mati listrik yang disebabkan bencana alam, kegagalan sistem, dan kesalahan manusia.
Pelajari selengkapnya tentang DR »

Arsip

Menghentikan infrastruktur fisik dan data arsip dengan S3 Glacier serta S3 Glacier Deep Archive. Kelas Penyimpanan S3 ini menahan objek jangka panjang pada laju yang paling rendah. Cukup buat kebijakan Siklus Aktif S3 untuk mengarsipkan objek melalui siklus aktifnya, atau unggah objek langsung ke kelas penyimpanan arsip. Dengan S3 Object Lock, Anda dapat menerapkan tanggal retensi ke objek guna melindungi mereka dari penghapusan, dan memenuhi persyaratan kepatuhan. Tidak seperti pustaka tape, S3 Glacier memungkinkan Anda memulihkan objek yang diarsipkan dalam satu menit untuk percepatan pengambilan dan 3-5 jam untuk pengambilan standar. Pemulihan data massal dari S3 Glacier dan semua pemulihan dari S3 Glacier Deep Archive selesai dalam 12 jam.
Pelajari selengkapnya tentang pengarsipan »

Data lake dan analisis big data

Mempercepat inovasi dengan membangun data lake di Amazon S3, dan mengekstrak wawasan penting dengan kueri terpasang, analitik, dan alat pembelajaran mesin. Seiring data lake Anda tumbuh, gunakan S3 Access Points untuk kemudahan mengkonfigurasi akses ke data Anda, dengan izin khusus untuk setiap aplikasi atau kumpulan aplikasi. Anda juga dapat menggunakan AWS Lake Formation untuk dengan cepat membuat data lake, dan menentukan secara terpusat dan menerapkan kebijakan keamanan, tata kelola, dan audit. Layanan tersebut mengumpulkan data di seluruh database dan sumber daya S3, memindahkannya ke data lake baru di Amazon S3, dan membersihkan dan mengklasifikasikannya menggunakan algoritme machine learning. Semua sumber daya AWS dapat diperbesar untuk mengakomodasi penyimpanan data yang diperluas — tanpa pelunasan investasi di awal.
Pelajari selengkapnya tentang membangun data lake di Amazon S3 dan AWS Lake Formation »

Penyimpanan cloud hybrid

Buat koneksi tanpa hambatan di antara aplikasi di lokasi dan Amazon S3 dengan AWS Storage Gateway guna mengurangi footprint pusat data Anda, dan memanfaatkan skala, keandalan, dan ketahanan AWS, serta machine learning dan kemampuan analitik AWS. Anda juga dapat mengotomatisasi transfer data antara penyimpanan lokal dan Amazon S3 dengan menggunakan AWS DataSync, yang dapat mentransfer data dengan kecepatan 10 kali lebih cepat dari alat sumber terbuka. Cara lain untuk mengaktifkan lingkungan penyimpanan cloud hybrid adalah untuk bekerja dengan penyedia gateway dari APN. Adda juga dapat mentransfer file langsung ke dalam dan keluar Amazon S3 menggunakan AWS Transfer for SFTP , layanan terkelola penuh yang memungkinkan pertukaran file aman dengan pihak ketiga.
Pelajari selengkapnya tentang penyimpanan hybrid, AWS DataSync, dan AWS Transfer for SFTP » 

Data aplikasi cloud-native

Membuat aplikasi seluler dan berbasis internet yang cepat dan hemat biaya dengan menggunakan layanan AWS dan Amazon S3 untuk menyimpan data produksi. Dengan Amazon S3, Anda dapat mengunggah sebesar apa pun jumlah data dan mengaksesnya di mana saja untuk menyebarkan aplikasi dan menjangkau lebih banyak pengguna akhir. Menyimpan data di Amazon S3 juga berarti Anda memiliki akses ke alat pengembang AWS terbaru dan layanan untuk analisis machine learning dan analisis untuk berinovasi dan mengoptimalkan aplikasi cloud-native.
Pelajari selengkapnya tentang aplikasi cloud-native »

Studi kasus

Netflix menawarkan jutaan jam konten dari Amazon S3 kepada pelanggan di seluruh dunia. Amazon S3 juga bertindak sebagai data lake untuk solusi analisis big data mereka.
FINRA menggunakan Amazon S3 untuk menyerap dan menyimpan data hingga lebih dari 75 miliar peristiwa pasar setiap harinya dan fungsi AWS Lambda untuk memformat dan memvalidasi data hingga lebih dari 200 aturan.
Airbnb mempercayakan data cadangan dan file statis pada Amazon S3, termasuk lebih dari 10 petabyte foto pengguna. Sebagai solusi yang lahir di cloud, mereka terus melakukan inovasi untuk menemukan cara baru menganalisis data yang disimpan di Amazon S3.
GE menggunakan Amazon S3 untuk menyimpan dan melindungi petabyte pada data pencitraan medis penting untuk layanan GE Health Cloud service mereka, yang menghubungkan ratusan dari ribuan mesin pencitraan dan perangkat medis lainnya.
