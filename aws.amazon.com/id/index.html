  • Live Nation Entertainment Berinovasi Lebih Cepat, Mengurangi Biaya, dan Meningkatkan Ketersediaan Aplikasi Menggunakan AWS

    Pelajari bagaimana perusahaan hiburan langsung di dunia menggunakan AWS Cloud untuk menghadirkan pengalaman yang ditingkatkan kepada pelanggannya dengan lebih cepat.
    Pelajari Selengkapnya 

    Mencari Pengalaman Pelanggan yang Lebih Baik

    Live Nation adalah pemimpin global dalam hiburan langsung yang memproduksi konser, menjual tiket, dan menghubungkan merek ke musik. Tahun 2016, Live Nation mengumumkan perpindahan infrastruktur IT globalnya ke AWS demi menghadirkan pengalaman yang lebih baik kepada penggunanya.

    Migrasi Mudah

    Perusahaan memindahkan 118 aplikasi dan 668 server ke AWS dalam 17 bulan tanpa menambah jumlah karyawan atau anggaran.

    Keuntungan Terobosan Bisnis

    Dengan pindah ke AWS, Live Nation telah bergerak dari memecahkan masalah perangkat keras ke menghadirkan berbagai ide inovatif yang melayani pelanggannya dengan lebih baik. Sejak implementasi, Live Nation mewujudkan 58 persen pengurangan dalam total biaya kepemilikan, mendukung 10 kali lebih banyak proyek dengan staf yang sama, dan mengupayakan 99 persen peningkatan dalam ketersediaan aplikasi.

    Peloton didirikan tahun 2012 oleh tim yang terdiri dari lima orang, dan diluncurkan di Kickstarter tahun 2013. Perusahaan ini lahir di AWS dan menghadirkan sepeda pertamanya tahun 2014.     Mendukung Komunitas Sesuai Skala

    Peloton menggunakan AWS untuk mendukung papan peringkat dalam kelas kebugaran sesuai permintaan dan live streamingnya, dan ini memerlukan pemrosesan real-time, latensi rendah, serta elastisitas tinggi untuk menghadirkan data pesepeda yang dapat dikustomisasi untuk komunitas beranggotakan lebih dari 1,4 juta yang bersepeda bersama-sama dari rumah.

    Meluncurkan Fitur Baru untuk Melayani Anggota dengan Lebih Baik

    Dengan AWS, Peloton dapat dengan cepat menguji dan meluncurkan fitur baru untuk meningkatkan pengalaman kebugaran komunitas di rumah yang unik.

    Membangun Fondasi di Cloud

    Epic Games telah menggunakan AWS sejak 2012 dan kini sepenuhnya menggunakan AWS Cloud, menjalankan armada server game di seluruh dunia, sistem platform backend, database, situs web, jalur analitik, dan sistem pemrosesan di AWS.

Dengan menggunakan AWS, Epic Games menyelenggarakan acara dalam game dengan ratusan juta pengguna yang diundang tanpa khawatir tentang kapasitas, memasukkan 125 juta acara per menit ke dalam saluran analitiknya, dan menangani pertumbuhan gudang data lebih dari 5 PB per bulan.

    Memberikan Pengalaman Gaming Terbaik

    Dengan AWS, Epic Games selalu meningkatkan pengalaman pemainnya serta menawarkan game dan elemen game yang baru dan menarik. Perusahaan berencana memperluas penggunaan layanan AWS di masa mendatang, termasuk machine learning dan layanan terkontainer.

  • Matson Mengoperasikan Pengiriman Global dan Bisnis Logistik di AWS

    Pelajari cara Matson menggunakan AWS untuk menggerakkan inovasi dan layanan pelanggan kelas-dunia, sekaligus mencapai keandalan, keamanan operasional dan penghematan biaya infrastruktur.
    Pelajari Selengkapnya 

    Pelacakan Container Real-Time

    Matson membangun aplikasi seluler flagship untuk pelacakan container global yang memungkinkan pelanggan melakukan pelacakan real-time pengiriman barang. Fitur berharga lain dalam aplikasi tersebut mencakup pencarian jadwal kapal yang interaktif, pencarian peta pangkalan berbasis lokasi, dan umpan kamera-gerbang langsung.

    Akses Perangkat Mobile

    Semua perangkat seluler mengakses AWS melalui Amazon API Gateway. Cara ini memberikan endpoint lokasi edge yang mudah ditemukan untuk akses ke dalam sumber daya pada virtual private cloud Matson yang sudah tersedia.

    Komputasi Tanpa Server

    Fungsi AWS Lambda dirancang menggunakan pola layanan-mikro dan dimodelkan di sekitar konteks bisnis berbasis-lautan khusus, seperti pelacakan kapal dan jadwal kapal. 

    Konfigurasi dan Penyimpanan Database

    Amazon DynamoDB mengelola konfigurasi juga konfigurasi umpan balik pengguna dan pemberitahuan umpan balik pengguna yang dikirimkan dari perangkat seluler. DynamoDB Streams memberikan notifikasi real-time ke tim layanan pelanggan Matson. 

    Pemantauan dan Peringatan Data

    Pelanggan Matson mengandalkan pelacakan container dan informasi status kapal akurat dan terbaru. Pemantauan dan peringatan kejadian sistem diarsipkan dengan menggunakan Amazon CloudWatch, Amazon SNS, Amazon SES, AWS Lambda, dan CloudWatch Logs. 

    Aplikasi Tanpa Server Ujung-ke-Ujung

    Matson sekarang dapat menawarkan aplikasi tanpa server ujung-ke-ujung kepada pelanggan untuk membantu melacak pengiriman, tanpa infrastruktur yang perlu dipelihara.

    Mengelola Aplikasi Bisnis yang Sangat Penting

    Organisasi IT BP mengelola aplikasi SAP yang digunakan ribuan karyawan di seluruh dunia untuk rantai suplai, pengadaan, keuangan, dan masih banyak lagi. 

    Meningkatkan Kecepatan & Kelincahan Biaya

    Untuk meningkatkan kecepatan dan mendapatkan kelincahan biaya, BP menggunakan Amazon EC2 untuk memindahkan aplikasi bisnis inti ini ke cloud. Sebagai tambahan, tim tersebut membangun instans EC2 X1 untuk meningkatkan skala dan untuk memberdayakan analitik real-time mereka.

    Peningkatan Kinerja

    Tim tersebut sekarang dapat mendirikan sistem sesuai permintaan dalam hitungan jam alih-alih dalam minggu atau bulan. BP melihat peningkatan kinerja di seluruh papan, mencakup 40 persen peningkatan kecepatan untuk sistem ERP Pelumas. 

    Pengamanan Data

    Sebagai bagian dari migrasi cloud, BP mengatur ulang standar keamanan menggunakan AWS Config, AWS Identity and Access Management (IAM), Amazon CloudWatch, dan AWS Trusted Advisor. Standar baru ini membantu BP mengembangkan kerangka kerja yang aman untuk pengoperasian organisasi IT.
