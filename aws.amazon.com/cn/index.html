  • Zillow 利用 AWS 转型房地产

    Zillow 帮助房地产行业走到线上，然后利用 AWS 云改造了房屋搜索流程。利用 AWS，Zillow 可向买方、卖方和承租人传递准确的房屋数据，让他们能够在快节奏的市场中做出明智的决定。
    了解更多 

    对速度和规模日益增长的需求

    当 Zillow 在接近 15 年前创建其房屋评估工具—Zestimate—的时候，已经开发了本地机器学习框架来处理批量的数据。但是，随着其流行性和复杂性升高，Zillow 需要通过更好的方法将 Zestimates 送入全国近 1 亿所房屋。

    更快、更加动态的房屋评估

    Zillow 将其 Zestimate 框架迁移到了 AWS，让其获得了接近实时地交付房屋评估信息的速度和规模。更加动态的评估能够更好地反映每所房屋的独有特点以及当地房屋市场的现状，因此客户在探索购买或出售流程时能够得到最新的数据。

    按照市场节奏接触客户

    在热门住房市场中，房屋从挂牌到出价可能只需要几天。Zillow 将 AWS 技术融入到其基础设施中，以快速而可靠地传送每月数以亿计的电子邮件，通知客户最新的挂牌房源、房屋状况以及其他信息。

    提升客户体验

    Live Nation 是全世界领先的现场娱乐公司，其业务包括举办音乐会、销售门票以及对接品牌和音乐等。2016 年，Live Nation 宣布将其全球 IT 基础设施迁移到 AWS，以向客户提供更好的体验。

    轻松迁移

    公司在 17 个月内就将 118 个应用程序和 668 个服务器迁移到 AWS，但没有增加任何人力或预算。

    革命性的效益

    通过迁移到 AWS，Live Nation 不再需要排查硬件故障，只需专注于能够更好服务客户的创新理念。从项目实施以来，Live Nation 的总体拥有成本降低了 58%，同一个员工可以支持的项目数量增加了 10 倍，应用程序可用性提高了 99%。

    在云上构建基础

    Epic Games 自 2012 年起一直使用 AWS，如今一切运营都在 AWS 云上进行，运行其全球游戏服务器队列、后端平台系统、数据库、网站、分析管道，并在 AWS 上处理系统。

    实时集装箱跟踪

    Matson 构建了一款旗舰版移动应用程序，用于跟踪全球集装箱，以便客户对其货件进行实时跟踪。该应用程序中的其他重要功能包括：交互式船舶时间表搜索、基于位置的港口地图查找和实时大门摄像头上传数据。

    移动设备访问

    所有移动设备都通过 Amazon API Gateway 访问 AWS。这为在 Matson 现有的虚拟私有云中访问资源提供了高度可用的边缘站点终端节点。

    无服务器计算

    AWS Lambda 函数使用微服务模式设计而成，并模拟了特定的基于海洋的业务环境，例如货件跟踪和船舶时间表。 

    数据库配置和存储

    Amazon DynamoDB 管理通用配置以及用户反馈配置和移动设备发送的用户反馈通知。DynamoDB Streams 向 Matson 的客户服务团队提供实时通知。 

    数据监控和提醒

    Matson 的客户依靠准确的最新集装箱跟踪和船舶状态信息。通过使用 Amazon CloudWatch、Amazon SNS、Amazon SES、AWS Lambda 和 CloudWatch Logs，实现了对系统事件的监控和提醒。 

    端到端无服务器应用程序

    Matson 现在可以为客户提供端到端无服务器应用程序，从而帮助跟踪他们的货件，且无需维护基础设施。
