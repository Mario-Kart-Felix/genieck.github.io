Amazon S3

Armazenamento de objetos construído para armazenar e recuperar qualquer volume de dados de qualquer local

Comece a usar o Amazon S3
Solicite mais informações

O Amazon Simple Storage Service (Amazon S3) é um serviço de armazenamento de objetos que oferece escalabilidade líder do setor, disponibilidade de dados, segurança e performance. Isso significa que clientes de todos os tamanhos e setores podem usá-lo para armazenar qualquer volume de dados em uma grande variedade de casos de uso, como sites, aplicações para dispositivos móveis, backup e restauração, arquivamento, aplicações empresariais, dispositivos IoT e análises de big data. O Amazon S3 fornece recursos de gerenciamento fáceis de usar, de maneira que você possa organizar os dados e configurar os controles de acesso refinados para atender a requisitos específicos comerciais, organizacionais e de conformidade. O Amazon S3 foi projetado para 99,999999999% (11 9s) de durabilidade e armazena dados para milhões de aplicativos para empresas de todo o mundo.

Benefícios

Performance, escalabilidade, disponibilidade e durabilidade líderes do setor

Escale os recursos de armazenamento para mais e para menos a fim de atender às demandas flutuantes, sem investimentos iniciais ou ciclos de compras de recursos. O Amazon S3 foi projetado para 99,999999999% (11 9s) de durabilidade de dados porque cria e armazena automaticamente cópias de todos os objetos do S3 em vários sistemas. Isso significa que os dados estão disponíveis quando necessário e protegidos contra falhas, erros e ameaças.
Saiba mais sobre durabilidade de dados do S3 »

Grande variedade de classes de armazenamento econômicas

Economize sem sacrificar o desempenho armazenando dados nas classes de armazenamento do S3, que dão suporte a níveis diferentes de acesso aos dados em taxas correspondentes. Use o S3 Storage Class Analysis para descobrir dados que precisam ser migrados para uma classe de armazenamento de custo menor com base em padrões de acesso e configurar uma política do S3 Lifecycle para executar a transferência. Também armazene dados com padrões de acesso desconhecidos ou em constante mudança no S3 Intelligent-Tiering, que classifica objetos com base em padrões de acesso mutáveis e oferece economia automaticamente.
Saiba mais sobre o S3 Storage Classes »

Recursos incomparáveis de segurança, conformidade e auditoria

Armazene os dados no Amazon S3 e os proteja contra acesso não autorizado com recursos de criptografia e ferramentas de gerenciamento de acesso. O S3 é o único serviço de armazenamento que permite que você bloqueie o acesso público a todos os seus objetos no bucket ou na conta com o S3 Block Public Access. O S3 mantém programas de conformidade como PCI-DSS, HIPAA/HITECH, FedRAMP, EU Data Protection Directive e FISMA para ajudar a cumprir requisitos normativos. A AWS também dá suporte a inúmeros recursos de auditoria para monitorar acesso a solicitações aos recursos do S3.
Saiba mais sobre a segurança do S3 e a sua conformidade »

Gerencie dados e acesse controles facilmente

O S3 oferece recursos robustos para gerenciar acesso, custo, replicação e proteção de dados. Os pontos de acesso do S3 facilitam o gerenciamento do acesso aos dados com permissões específicas para suas aplicações usando um conjunto de dados compartilhados. Os recursos de replicação do S3 gerenciam a replicação de dados dentro de uma região ou com outras regiões. As operações em lote do S3 ajudam a gerenciar mudanças de grande escala entre bilhões de objetos. Como o S3 funciona com o AWS Lambda, é possível registrar atividades, definir alertas e automatizar fluxos de trabalho sem gerenciar infraestrutura adicional.
Saiba mais sobre os recursos de gerenciamento do armazenamento do S3 »

Serviços de consulta no local para análises

Execute análises de big data nos objetos do S3 (e em outros conjuntos de dados na AWS) com os serviços de consulta no local. Use o Amazon Athena para consultar dados do S3 com expressões SQL padrão e o Amazon Redshift Spectrum para analisar dados armazenados nos data warehouses e recursos do S3 na AWS. Você também pode usar o S3 Select para recuperar subconjuntos de dados do objeto, em vez de todo o objeto, e aumentar a performance das consultas em até 400%.
Saiba mais sobre a consulta no local »

Serviço de armazenamento em nuvem mais aceito

Armazene e proteja os dados no Amazon S3 trabalhando com um funcionário da AWS Partner Network (APN) – a maior comunidade de fornecedores de serviços de nuvem de tecnologia e consultoria. A APN reconhece parceiros de migração que transferem dados para o Amazon S3 e parceiros de migração que oferecem soluções integradas do S3 para armazenamento principal, backup e restauração, arquivamento e recuperação de desastres. Também compre uma solução integrada à AWS diretamente no AWS Marketplace, que lista mais de 250 opções específicas de armazenamento.
Saiba mais sobre o APN e o AWS Marketplace »

Como os pontos de acesso do S3 funcionam

Os pontos de acesso do Amazon S3 simplificam o gerenciamento do acesso de dados em grande escala para aplicativos que usam conjuntos de dados compartilhados no S3. Com os pontos de acesso do S3, agora você pode criar centenas de pontos de acesso por bucket, representando uma nova maneira de fornecer acesso a conjuntos de dados compartilhados. Os pontos de acesso fornecem um caminho personalizado em um bucket, com um hostname e uma política e acesso únicos, que aplica as permissões e os controles de rede específicos para qualquer solicitação feita por meio do ponto de acesso.

Casos de uso

Backup e restauração

Crie soluções escaláveis, duráveis e seguras de backup e restauração com o Amazon S3 e outros serviços da AWS, como o S3 Glacier, o Amazon EFS e o Amazon EBS, para aumentar ou substituir recursos existentes no local. Os parceiros da AWS e da APN podem ajudar a atender às exigências de Recovery Time Objectives (RTO – Objetivos do tempo de recuperação), Recovery Point Objectives (RPO – Objetivos do ponto de recuperação) e de conformidade. Com a AWS, faça backup dos dados já na Nuvem AWS ou use o AWS Storage Gateway, uma serviço de armazenamento híbrido, para enviar backups de dados no local para a AWS.
Saiba mais sobre backup e restauração »

Disaster Recovery (DR – Recuperação de desastres)

Proteja dados, aplicativos e sistemas de TI críticos que estejam em execução na Nuvem AWS ou em sua organização sem incorrer na despesa de um segundo local físico. Com armazenamento do Amazon S3, o S3 Cross-Region Replication e outros serviços de computação, redes e banco de dados da AWS, crie arquiteturas de Recuperação de desastres para se recuperar rápida e facilmente de interrupções causadas por desastres naturais, falhas no sistema e erros humanos.
Saiba mais sobre Recuperação de desastres »

Arquivamento

Desative infraestrutura física e arquive dados com o S3 Glacier e o S3 Glacier Deep Archive. Essas categorias de armazenamento do S3 mantêm objetos por longos prazos com as taxas mais baixas. Basta criar uma política de ciclo de vida do S3 para arquivar objetos ao longo de todos os seus ciclos de vida ou fazer upload de objetos diretamente nas categorias de armazenamento de arquivamento. Com o S3 Object Lock, implante datas de retenção em objetos para protegê-los contra exclusões e atender às exigências de conformidade. Diferentemente das bibliotecas de fitas, o S3 Glacier permite restaurar objetos arquivados em menos de um minuto para recuperações rápidas e de 3 a 5 horas para recuperações padrão. Os dados em massa são restaurados pelo S3 Glacier e todas as restaurações pelo S3 Glacier Deep Archive são concluídas em 12 horas.
Saiba mais sobre arquivamento »

Análises de data lakes e big data

Agilize a inovação criando um data lake no Amazon S3 e extraia insights valiosos usando ferramentas de consulta no local, análises e ferramentas de machine learning. A medida que seu data lake cresce, use os pontos de acesso do S3 para configurar facilmente o acesso a seus dados, com permissões específicas para cada aplicativo ou conjunto de aplicativos. Use também o AWS Lake Formation para criar rapidamente um data lake, bem como definir e aplicar de forma centralizada políticas de segurança, governança e auditoria. O serviço coleta dados nos bancos de dados e nos recursos do S3, migra esses dados para um novo data lake no Amazon S3, e os limpa e classifica usando algoritmos de machine learning. A escala de todos os recursos da AWS pode ser aumentada para acomodar datastores em expansão, sem investimentos iniciais.
Saiba mais sobre a criação de um data lake no Amazon S3 e o AWS Lake Formation »

Armazenamento em nuvem híbrida

Crie uma conexão perfeita entre os aplicativos no local e o Amazon S3 com o AWS Storage Gateway para reduzir o espaço do datacenter e utilizar a escala, a confiabilidade e a durabilidade da AWS, bem como os recursos inovadores de machine learning e análises. Também automatize transferências de dados entre o armazenamento no local e o Amazon S3 usando o AWS DataSync, que pode transferir dados em velocidades até 10 vezes maiores do que ferramentas de código aberto. Outra maneira de permitir um ambiente de armazenamento na nuvem híbrida é trabalhando com um fornecedor de gateway da APN. Também transfira arquivos diretamente para dentro e para fora do Amazon S3 com o AWS Transfer for SFTP – um serviço totalmente gerenciado que permite trocas de arquivos seguras com terceiros.
Saiba mais sobre armazenamento híbrido, AWS DataSync e AWS Transfer for SFTP » 

Dados de aplicativos nativos da nuvem

Crie aplicativos para dispositivos móveis e baseados na Internet rápidos e econômicos usando os serviços da AWS e o Amazon S3 para armazenar dados da produção. Com o Amazon S3, faça upload de qualquer volume de dados e o acesso de qualquer lugar para implantar aplicativos mais rapidamente e alcançar mais usuários finais. O armazenamento de dados no Amazon S3 também significa que você tem acesso às ferramentas e aos serviços de desenvolvedor da AWS mais recentes de machine learning e análise para inovar e otimizar os aplicativos nativos da nuvem.
Saiba mais sobre aplicativos nativos da nuvem »

Estudos de caso

A Netflix oferece bilhões de horas de conteúdo do Amazon S3 para clientes em todo o mundo. O Amazon S3 também funciona como o data lake da solução de análises de big data.
O FINRA usa o Amazon S3 para processar e armazenar dados de mais de 75 bilhões de eventos de mercado diariamente, e o AWS Lambda funciona para formatar e validar os dados em relação a mais de 200 regras.
O Airbnb armazena dados de backup e arquivos estatísticos no Amazon S3, inclusive mais de 10 petabytes de fotografias de usuários. Como uma solução concebida na nuvem, eles criam continuamente maneiras inovadoras de analisar dados armazenados no Amazon S3.
A GE usa o Amazon S3 para armazenar e proteger um petabyte de dados de diagnóstico por imagens médicas críticas para o serviço GE Health Cloud, que conecta centenas de milhares de máquinas de diagnóstico por imagem e outros dispositivos médicos.
